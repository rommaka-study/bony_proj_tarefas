# Projeto: To-do list simples com HTML, CSS e Javascript


#DESCRIÇÃO GERAL COMO O PROJETO VAI FUNCIONAR

Este será um projeto de tela unica, em que na parte superior teremos um campo de texto para adicionar uma tarefa nova e um botão de envio

Ao envia a tarefa, ela será adicionada a uma lista exibida logo abaixo desse campo de texto.


#DETALHES TÉCNICOS

As tarefas serão armazenadas em um array no proprio javascript, sem uso de nenhuma ferramanta ou sistema externo

Cada tarefa:
- Texto
- Boolean confirmando se está feita ou não

Sobre as ações possíveis de cada tarefa:
- marcar/desmarcar como concluida, daí o texto fica riscado (via classe css)
- Deletar uma tarefa, então ela some da lista completamente.
- Adicionar uma tarefa nova através do campo de texto na parte superior da tela, que será sempre adicionada com o boolean de confirmação em false.

#INSTRUÇÃO DE ORGANIZAÇÃO DO CODIGO

O projeto será feito utilizando HTML, CSS e Javascript simples.

No Javascript, trabalhar sempre com EventListerners e não adicionando eventos no HTML diretamente.

No inicio do Javascript, já selecione em variáveis todos os elementos que serão manipulados


#INTERFACE GRAFICA

- O fundo do pojeto será cinza claro.
- Tatno o campo quanto a lista ficarão em cards separados, com fundo pranco,borda arrendondada e uma leve separação entre os cards.

A cor predominante para os botões(como o botão de adicionar e deletar) será
verde escuro.
